## Overview
A full investment banking-style equity research and valuation model built on 
Bharat Electronics Limited (BEL) — a Navratna Defence PSU under the Ministry 
of Defence, Government of India. The model replicates a sell-side equity 
research report covering historical analysis, forecasting, and multi-methodology 
valuation across FY2021–FY2035.

## 📁 Model Structure (20 Sheets)
| Sheet | Description |
|---|---|
| Assumptions | Bull / Base / Bear scenario inputs, key drivers |
| Historical IS | 5-year Income Statement (FY2021–FY2025) |
| Historical BS | 5-year Balance Sheet (FY2021–FY2025) |
| Historical CF | 5-year Cash Flow Statement (FY2021–FY2025) |
| Beta | Regression-based Beta using monthly returns vs Nifty |
| Market Data | Live market inputs, price, shares outstanding |
| WACC | Cost of Equity (CAPM), Cost of Debt, WACC calculation |
| Forecasts & Support Schedules | 10-year forecasted P&L, BS, CF (FY2026–FY2035) |
| Ratio Analysis | Operational, Profitability, DuPont, Valuation ratios |
| Common Size | Vertical analysis of P&L and BS |
| Comps Valuation | Peer benchmarking across 7 Indian defence companies |
| DCF Valuation & Sensitivity | DCF model with 2x sensitivity tables |
| Profit & Loss | Formatted standalone P&L output |
| Balance Sheet | Formatted standalone BS output |
| Cash Flow | Formatted standalone CF output |
| Quarters | Quarterly financial trends |
| Summary | Football field valuation summary |
